# twiQuery

![MIT](https://img.shields.io/badge/license-MIT-blue.svg)

Tweets scraper enables you to do advanced search for tweets like advanced search twitter web page, save the collected tweets.
 
 Supported output format:
 CSV,
 JSON,
 SQLite
